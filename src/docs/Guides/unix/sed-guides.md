# SED学习总结

SED编译器被称为流编辑器。可以在处理数据之前基于事先提供的一组规则遍历数据流。可以包括文本文件中的搜索关键字全局替换、局部替换（特定行，和标记）、删除行、添加或追加行；处理单个字符的转换
命令（y）

  - SED命令格式
  ```sh
     sed options program file
  ```
  options常用的可以包括如下表：
  
| 选项 | 描述 |
| :----: | :----: |
| -F  | 读取行后划分字段的分隔符 |
| -f  | 执行sed文件中的命令集合 |
| -n  | 禁止sed编译器的输出,对比P命令|
| -v var = value  | 设置变量并设置默认值 |
