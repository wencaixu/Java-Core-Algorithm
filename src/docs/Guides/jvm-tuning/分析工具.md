## JVM调优之jstack命令工具
### 概述:
   打印Java线程，核心文件或者远程调试服务的堆栈信息

### 命令格式：

```shell
jstack [options] pid
jstack [options][server-id@]remote-ip
```

说明：pid是追踪的堆栈信息的Java进程id，Java进程可以使用JPS命令进行查看

### 参数：

| Options  | 描述 |
| -------- | ---- |
| -F       |      |
| -l       |      |
| -m       |      |
| -h/-help |      |

**reference**: https://docs.oracle.com/javase/8/docs/technotes/tools/windows/jstack.html#BABGJDIF
